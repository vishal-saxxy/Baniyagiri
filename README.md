 Baniyagiri

The Zero-Friction Expense Splitter

Baniyagiri is a local-first expense management application designed for groups, trips, shared households, and events. It enables users to track expenses, split costs using multiple methods, and generate simplified settlements without requiring accounts, authentication, or external services.

 Features

* Multi-workspace expense management
* Occasion-based expense organization
* Equal, Unequal, Percentage, Shares, and Adjustment splits
* Real-time balance calculations
* Debt simplification engine for optimized settlements
* Expense editing and recalculation
* Analytics dashboard and spending insights
* Offline-first architecture
* Local data persistence using Zustand Persist and localStorage
* Mobile-optimized user experience

 Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Zustand

 Privacy

Baniyagiri operates entirely within the browser. All data is stored locally on the user's device. No accounts, servers, cloud storage, analytics, or third-party services are used.
